Create a linear color map.

Parameters
v_mindouble minimum value to be represented on the color map
v_maxdouble maximum value to be represented on the color map
color_minColor Color at the bottom of the scale
color_maxColor Color at the top of the scale
number_of_levelsint number of levels between v_min and v_max
Returns
linear_color_map std::map<double,Color> a map associating double values in the interval \( [v_{min},v_{max}] \) to a color linearly interpolated between color_min and color_max

This function creates a linear color map associating the color color_min to v_min and the color color_max to v_max, dividing the interval \([v_{min},v_{max}]\) into number_of_levels sub-intervals and associating a color in each sub-interval by interpolating colors linearly between color_min and color_max.
